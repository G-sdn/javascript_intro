This html document outlines the source code for a very brief introduction to JavaScript in preparation to creative coding using p5.js.

To check out the JavaScript code, please have a look into the project folder and examine the scripts.
In order to execute the JavaScript code of the individual scripts, click the buttons beneath accordingly.
Please open your web browser's console to follow the results of executing the scripts.

  • Chrome: From the menu bar, select View - Developer - JavaScript Console.
  • Firefox: From the menu bar, select Tools - Web Developer - Web Console.
  • Safari: From the menu bar, select Develop - Show Web Inspector - and select the Console tab.
